# Druk Wide Cyr (self-hosted)

В `src/index.css` объявлены `@font-face` для весов **100–900** (шаг 100), файлы:

`DrukWideCyr-{100|200|300|400|500|600|700|800|900}.woff2`

## Лицензия и замена файлов

Druk Wide Cyr — коммерческий шрифт. Файлы с одинаковым размером для разных весов сейчас — **копии Regular (400)**; замените их на **woff2** из вашего лицензионного комплекта, сохранив имена или обновив пути в `index.css`.

## Конвертация

Из `.otf` / `.ttf` можно собрать woff2 (например `woff2_compress` из [google/woff2](https://github.com/google/woff2)).
